## 🚀 What this is

wecom-homework-auto-tracker helps you manage homework tracking with a simple dashboard. It uses student IDs instead of names, so you can review progress with less personal data on screen.

This app is built for Windows users who want a local tool for homework records and a web dashboard for quick checks.

## 🔍 Troubleshooting

### App does not open

- Check that the download finished
- Make sure you extracted the `.zip` file if needed
- Try running the `.exe` file again
- Right-click the app and choose Run as administrator if Windows blocks it

### Browser page does not load

- Wait a few seconds and refresh the page
- Close other browser tabs and try again
- Check that the app is still running
- Make sure the dashboard address is correct

### Data does not appear

- Confirm that your file is in the right folder
- Check that the file format matches what you imported
- Make sure the student ID list is correct
- Reload the app after changing the source file

### Students show up with wrong IDs

- Review your ID mapping file
- Check for extra spaces in the ID column
- Make sure each ID is unique
- Replace old data with the latest version
